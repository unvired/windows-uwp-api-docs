DataManagerImpl Class

DataManagerImpl provides access to the application's database. DataManagerImpl provides methods to get, insert, modify and delete data into / from the database. DataManagerImpl deals with data in terms of C# Objects - DataStructure. DataManagerImpl also allows execution of custom SQL statements. All methods which allow execution of custom SQL statements start with 'execute'.

How to get access to IDataManager: Use ApplicationManager to get access to the IDataManager.

Examples
Copy
ApplicationManager applicationManager = ApplicationManager.getInstance();
DataManagerImpl dataManager = applicationManager.getDataManager();
SQLite is used as the underlying database.
